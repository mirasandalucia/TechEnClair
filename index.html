MALIN

Mini-PC Occasion

80€ à 120€

Dell OptiPlex / HP ProDesk. Config cible : i3/i5, 8Go RAM, SSD 120Go. C'est le meilleur rapport puissance/prix actuel.

HA Green

~100€

Zéro prise de tête. Tu branches, ça marche. Solution officielle, stable, mais limitée pour la vidéo.

HA Yellow

150€+

Zigbee et Matter intégrés, slot SSD NVMe. C'est le serveur tout-en-un propre et officiel.

Raspberry Pi 4 / 5

130€ à 150€

Le classique compact. Attention, le prix du kit complet dépasse souvent celui d'un Mini-PC.

Freebox Delta/Ultra

0€ (Inclus)

Utilise les VM de ta box. Parfait pour un premier test gratuit avant d'investir.

Le Choix Zigbee

30€ - 35€

Indispensable. Prends un Sonoff Dongle-E ou un SkyConnect pour tes capteurs.

⚠️ Pas de carte SD pour le long terme !

Installer HACS proprement

0 Installation de SSH

1. Va dans Paramètres > Modules.

2. Cherche "Terminal & SSH" dans la boutique.

3. Installe et clique sur Démarrer.

1 La Commande

wget -O - https://get.hacs.xyz | bash -

2 Le Reboot

Redémarrage complet : Paramètres > Système > Redémarrer.

3 Activer HACS

Paramètres > Appareils et services > Ajouter une intégration > HACS.

4 GitHub

Liaison GitHub avec le code à 8 chiffres.
