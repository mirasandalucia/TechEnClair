Bien choisir son
Serveur Home Assistant

Ton serveur est le cœur du système. Un mauvais choix = bugs et lenteurs. On fait le point proprement sur le matériel indispensable en 2026.

Quel matériel utiliser pour
Home Assistant ?

MALIN

Mini-PC Occasion

80€ à 120€

Dell OptiPlex / HP ProDesk. Config cible : i3/i5, 8Go RAM, SSD 120Go. C'est le meilleur rapport puissance/prix actuel.

Home Assistant Green

~100€

Zéro prise de tête. Tu branches, ça marche. Solution officielle, stable, mais limitée si tu veux faire de la reconnaissance vidéo.

Home Assistant Yellow

150€+

Zigbee et Matter intégrés, slot SSD NVMe. C'est le serveur tout-en-un pour ceux qui veulent une installation propre et officielle.

Raspberry Pi 4 / 5

130€ à 150€

Le classique compact. Attention, le prix du kit complet (boitier+alim+SSD) dépasse souvent celui d'un Mini-PC plus puissant.

Freebox Delta / Ultra

0€ (Inclus)

Utilise les VM (machines virtuelles) de ta box. Parfait pour un premier test gratuit avant d'investir dans une machine dédiée.

Le Choix Zigbee

30€ - 35€

Indispensable. Prends un Sonoff Dongle-E ou un SkyConnect pour piloter tes ampoules et capteurs sans dépendre du cloud.

⚠️ Rappel : Pas de carte SD pour le long terme !

Installer HACS proprement

0 Installation de SSH

1. Va dans Paramètres > Modules complémentaires.

2. Clique sur Boutique des modules complémentaires (en bas à droite).

3. Cherche "Terminal & SSH". Clique sur installer.

4. Active "Afficher dans la barre latérale" et clique sur Démarrer.

1 La Commande Magique

Ouvre le Terminal dans ta barre latérale et colle ceci :

wget -O - https://get.hacs.xyz | bash -

2 Le Reboot Obligatoire

HACS ne peut pas s'activer sans un redémarrage complet du cœur.
Va dans Paramètres > Système > Redémarrer.

3 Activer l'Intégration

Une fois redémarré :

  • • Va dans Paramètres > Appareils et services.
  • • Clique sur Ajouter une intégration.
  • • Cherche HACS et coche les 4 cases obligatoires.

4 Liaison GitHub

Un code à 8 chiffres apparaît. Clique sur le lien, connecte ton GitHub et valide le code.

⚠️ Astuce : Si HACS ne s'affiche pas à gauche, fais CTRL+F5 sur ton clavier.
