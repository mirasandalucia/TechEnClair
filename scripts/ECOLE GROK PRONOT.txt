Grok analyse a la fin des cours PRONOT Les Devoir, les punitions et les retards de la journée.

---------------------------------------------------------------------

voici le code automatisation a modifier : CALENDAR ! AGEND ID ! SENSOR PRONOT ! MOBILE PARENTS NOTIFY


------------------------------------------------------------------


alias: "Suivi École Esteban via Grok"
description: "Envoie un résumé des devoirs et incidents (sans les absences) à la fin des cours"
trigger:
  - platform: calendar
    event: end
    entity_id: calendar.esteban
condition:
  - condition: time
    after: "15:00:00"
action:
  - service: conversation.process
    data:
      agent_id: conversation.grok
      text: >
        Analyse ces données Pronote pour Esteban et fais un résumé court aux parents :
        - Devoirs : {{ states('sensor.pronote_miras_esteban_homework') }}
        - Retards : {{ states('sensor.pronote_miras_esteban_delays') }}
        - Punitions : {{ states('sensor.pronote_miras_esteban_punishments') }}
        S'il n'y a rien de spécial, dis simplement que la journée s'est bien passée.
    response_variable: grok_output

  - service: notify.mobile_app_sm_s921b
    data:
      title: "📝 Point école Esteban"
      message: "{{ grok_output.response.speech.plain.speech }}"

  - service: notify.mobile_app_sm_julie
    data:
      title: "📝 Point école Esteban"
