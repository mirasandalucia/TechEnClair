1er - Site IA : https://www.hedra.com/

2e - crée sont avatar ou importer sont avatar

3e - On crée la vidéo avec un prompt (voir ma vidéo)

4 - transformer son fichier mp4 to Webp avec ffmpeg.exe ( le placer dans le dossier ou il y a l'image ) Clique droit ouvrir dans le Terminal

coller le script suivant : 
.\ffmpeg.exe -i TonFichier.mp4 -vcodec libwebp -vf "fps=17,scale=512:-2" -lossless 0 -compression_level 6 -q:v 50 -loop 0 -preset picture -an avatar_anime_q50.webp

(Remplacer Tonfichier.mp4 par le votre).

5e - Aller sur le site https://ezgif.com/webp-to-gif pour transformer votre fichier Webp to GIF
